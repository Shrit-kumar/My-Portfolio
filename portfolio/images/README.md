# Images folder

Add your real photos here with these exact filenames (the site already links to them):

| Filename          | Used for                                  | Recommended size       |
|--------------------|--------------------------------------------|--------------------------|
| `profile.jpg`      | Hero section photo (inside the glass card) | 960×1200px, portrait, JPG/PNG/WEBP |
| `og-cover.png`     | Social media preview image (WhatsApp, LinkedIn, Twitter) | 1200×630px |

Until you add `profile.jpg`, the hero card will automatically show a clean
"SK" placeholder instead of a broken image icon — so the site still looks
finished if you forget this step before deploying.

Tip: export your photo with decent contrast against a dark background —
since the card sits on `#0A0A0A`, a plain or slightly blurred background in
the photo will blend in nicely with the glassmorphism frame.
